
# Othello Simulator

```
$ ./othello.py --help
usage: othello.py [-h] [--delay DELAY] [--service SERVICE] [--hosts HOSTS]
                  [--branch BRANCH] [--depth DEPTH]

optional arguments:
  -h, --help         show this help message and exit
  --delay DELAY      Communication delay between network elements (ns)
  --service SERVICE  Service time at each host (ns)
  --hosts HOSTS      Number of hosts to use in the simulation
  --branch BRANCH    Othello game tree branching factor
  --depth DEPTH      How deep to search into the game tree
```
